# 🎬 Movie Recommendation System

## 📌 Overview
This project builds a movie recommendation system using collaborative filtering.  
It recommends movies based on user rating patterns by identifying similar movies using cosine similarity.

---

## 🎯 Problem Statement
With a large number of movies available, users need personalized recommendations.  
This project aims to suggest relevant movies based on a user's past preferences.

---

## ⚙️ Approach

1. Loaded MovieLens dataset
2. Performed Exploratory Data Analysis (EDA)
3. Created a user-movie matrix
4. Filled missing values for similarity computation
5. Calculated movie similarity using cosine similarity
6. Generated recommendations:
   - Similar movies
   - Personalized user recommendations
